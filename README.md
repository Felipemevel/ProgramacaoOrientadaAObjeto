# Programação Orientada a Objetos (POO)

Repositório reservado para os estudos e práticas da disciplina de Programação Orientada a Objetos do Instituto Federal do Rio Grande do Norte (IFRN).

O objetivo deste espaço é documentar o meu progresso e aprendizado na matéria, centralizando a resolução de atividades teóricas e práticas. Aqui estarão armazenados os códigos de listas de exercícios da disciplina, bem como desafios de plataformas de programação competitiva (como o Beecrowd).

##  Linguagem Utilizada

* **Java**: Linguagem principal utilizada para a aplicação dos conceitos de POO e resolução de todos os problemas.
